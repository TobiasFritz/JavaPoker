# JavaPoker

## Requirements:
- Java v17.0.2
- JUnit v5.7.0

## How to run:
Navigate to source files and run `javac Poker.java` to compile.
Run `java -cp . Poker` to start the program.

Console output should read:

    Poker start
    
    - hand 1: [DA, DK, DQ, DJ, DT]
    - hand 2: [S6, S5, S4, S3, S2]

    Winning hand: [DA, DK, DQ, DJ, DT]
